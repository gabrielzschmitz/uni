\section{Introdução}

\paragraph{}
Esta atividade teve como objetivo realizar testes de segurança em uma API
vulnerável, simulando um cenário real de Pentest em aplicações modernas.
Durante os exercícios foram analisadas diferentes vulnerabilidades
frequentemente encontradas em APIs, incluindo exposição excessiva de dados,
falhas de autorização, SQL Injection, Mass Assignment e alteração não
autorizada de senhas.

\paragraph{}
As atividades foram executadas utilizando a máquina \texttt{atacante} contra o
ambiente disponibilizado para o laboratório. Foram utilizadas ferramentas como
\texttt{curl} e \texttt{sqlmap} para exploração e validação das
vulnerabilidades encontradas.
